# Smarter_School (Backend)

## Project Overview
 Smarter_School  is a backend automation system built with Django and Django REST Framework.  
It is designed to help African schools manage students, attendance, results, and parent communication efficiently using a centralized API-driven platform.

This project is developed as part of the **ALX Backend Engineering Capstone Project**.

---

## Features (MVP)

### Authentication & Authorization
- JWT-based authentication (Simple JWT)
- Role-based access control:
  - Admin
  - Teacher
  - Parent

### Student Management
- Create and manage student records
- Assign students to classes
- Automatic promotion logic
- Student history tracking

### Attendance System
- Daily attendance marking
- Attendance reports
- Absentee tracking

### Result & CA Automation
- Continuous Assessment (CA) and exam score entry
- Automatic total score calculation
- Automatic grade assignment
- Parent access to student results

### Parent Communication
- Notification triggers (results upload, attendance updates)
- Designed for future SMS/email integration

---

## Technology Stack
- **Backend:** Django, Django REST Framework
- **Database:** PostgreSQL (Render)
- **Authentication:** Simple JWT
- **Deployment:** Render
- **Version Control:** Git & GitHub
